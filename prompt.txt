I want to create a personalized time management system for my own use.

The system should have the following pages

1. Home Tasks
2. Work Tasks
3. Free Time Tasks
4. Generate Block

The Home Tasks page should have the ability to add, edit, and delete home tasks. There are several different types of home tasks. Repeating Tasks, Priority Tasks, Scheduled Tasks, Misc Tasks.
Tasks can also be marked as In Progress or Blocked. Blocked Tasks Require a Blocked Reason.

The Work Tasks page should have the ability to add, edit, and delete work tasks. For work tasks, each task is a work item, and each of those work items can have subtasks associated with it. 
There are two sections for work tasks, Active Work Items and Backlog. 
Active work Items are currently being worked on and backlogged work items are waiting to go into the active section when there is room. 
There can be up to 5 active work items at one time. 
This tasks in the backlog should be able to be moved into the active section and vice versa. 

The Free Time Tasks Page should have the ability to add, edit, and delete free time tasks. 
For free time tasks, there can be different categories, tasks and two tasks in each category can be active at a given time. 
The categories now are Gaming, Brewing, Building, Artwork, Reading, and Music. 
There should be an active section and a backlog section, each divided out by category, with the backlog section being backlogged free time tasks in each category and the active section being currently active tasks for each category. 

The Generate Block page should generate a block of tasks, one home task, one work task, and one free time task.
For the home task it should pick a random task from a category depending on priority. Scheduled (If Today), Priority, In Progress, Repeating, Misc
For the work task it should randomly pick an active work item, and then pick a subtask from that work item.
For Free time it should randomly pick one of the active tasks (there are 2 in each active category).
On the generate block page I should be able to reroll each task type, and each task in the block should be able to be marked as complete, mark as in progress, or marked as blocked. 
For the home task there should be a button to move on to the next category. Also when home task is generated if a category is empty it should skip it, moving to the next priority category. The button should be disabled when on misc. 
When they are marked they are removed from the block. Up to 3 blocks can be generated and worked on at a time. 
I should also have the option to instead of randomizing, manually select tasks to replace the randomized ones.

On the home tasks page there should be a button for reset weeklies, reset monthlies, reset quarterlies, and reset yearlies. In the repeating section it should be organized by repeating type (weekly, monthly, etc).
When tasks are marked as complete and are repeating tasks, they should not be able to be rolled, but remain in completed status. When I click reset X it should reset those tasks to incomplete, allowing them to be rolled again. 

There should be an import and export button to import and export all data to be used on another device should it be needed. That way I can swap between mobile and desktop even with different browsers. 

This should be a web app that I can deploy and run on my own domain.
